Was ist PitBike OS?

PitBike OS ist ein selbst entwickeltes Mikrocontroller-System für Pitbikes, das Telemetrie, Elektronik-Steuerung, Sensoren und ein digitales Dashboard vereint. Es soll leicht zu installieren, zu erweitern und zu bedienen sein.

Hauptfeatures

Dashboard

Digitale Anzeige für Geschwindigkeit, Drehzahl, Temperatur und mehr.

Sensor-Support

Support für Temperatur, Spannung, Neigung und Feuchtigkeitssensoren.

OTA-Updates

Software-Updates kabellos über WLAN durchführen.

Bluetooth-App

Steuerung und Live-Daten in einer Android-App (Beta).

Erweiterbar

Eigenen Code, Module und Widgets einfach hinzufügen.

Open Source

Das gesamte Projekt ist offen auf GitHub einsehbar.

Installation

Lade einfach den neusten Build aus dem GitHub-Repository herunter und flashe ihn auf deinen ESP32 bzw. Controller.
